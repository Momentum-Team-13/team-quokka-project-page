Project Description

This is a social greeting card application built using Django REST Framework and React. Users can create greeting cards that are viewed by users that follow them. When a user logs in, they will first see their timeline, a list of all cards created by users they follow in descending chronological order (newest to oldest). The user can also choose to view their profile, a list of all cards they have created, or all cards created by another user. A list of all cards created on the site is available to explore.

From the list of all cards or the list of all users, a user can select another user, then select to follow that user. This will add that user‚Äôs cards to the first user‚Äôs timeline.
